# 搭建Spigot端服务器
## 概述：
本篇将教会您搭建Spigot端服务器

## Spigot简介
Spigot简介资料出处：https://www.bilibili.com/read/cv18010317/

Spigot服务端，是基于Minecraft原版服务端改进的插件服务端，大家可在Spigotmc社区上找到众多Minecraft第三方插件，Spigot也是目前开服人数最多的服务端之一，当然也可在国内我的世界中文论坛下找到国人原创插件和搬运插件，Spigot不支持Mod，仅支持Plugins。
